Given z is true and x || (!y && z) is false,
what must be the boolean value of y?

ACCESS GRANTED

Both x and (!y && z) must be false if x || (!y && z) is false. (!y && z) is false when at least one of the clause is false. Since z is true, !y must be false, and so y is true.

ACCESS DENIED

Both x and (!y && z) must be false if x || (!y && z) is false. (!y && z) is false when at least one of the clause is false. Since z is true, !y must be false, and so y is true.
